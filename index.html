Мои работы

Fire sensors demo

http://fire.korbit.eu

Fire sensor demo это проект, который связан с сенсорами. Как только один из датчиков присылает состояние отличное от нормального на экране появляется сообщение об угрозе. Проект был сделан на Angular.

AppyBuy

https://appybuy.de/de/index

AppyBuy с одной стороны это мессенджер имеющий админитративную панель. С другой стороны этот проект сотрудничает со многими магазинами. Деля покупки через сайт, некоторый процент идет в футбольный клуб где занимается ваш ребенок. AppyBuy работает пока только на немецкой площадке. Проект сделан на Magnolia.

etm-system

Frontend часть проекта сделана на фрэймворке Vue. Задачей проекта является продажа ЖД билетов. Проект еще находится в стадии разработки.
